Інструкція константи 16b та додавання:

| С код                                             | MIPS код                         |
| ------------------------------------------------- | -------------------------------- |
| // int is a 32-bit signed word<br>int a = 0x4f3c; | #$s0 = a<br>addi $s0, $0, 0x4f3c |
• Константа 32b та зчитування 16 старших бітів за допомогою lui ("load upper immediate") та виконання інструкції ori:

| С код               | MIPS код                                             |
| ------------------- | ---------------------------------------------------- |
| int a = 0xFEDC8765; | # $s0 = a<br>lui $s0, 0xFEDC<br>ori $s0, $s0, 0x8765 |
