# Пам'ять
В пам'яті зберігаються:
- Інструкції
- Дані
	- Глобальна/статична: виділяється перед запуском програми
	- Динамічна: виділяється під час виконання програми
- Який розмір пам'яті?
	- 232 = 4 гігабайти (4 ГБ)
	- Від адреси 0x00000000 до 0xFFFFFFFF

# Приклад
```C
int f, g, y; // global variables
int main(void){
	f = 2;
	g = 3;
	y = sum(f, g);
	return y;
}

int sum(int a, int b){
	return (a + b);
}
```

```MIPS
data
f:
g:
y:
.text
main:
	addi $sp, $sp, -4 # stack frame
	sw $ra, 0($sp) # store $ra
	addi $a0, $0, 2 # $a0 = 2
	sw $a0, f # f = 2
	addi $a1, $0, 3 # $a1 = 3
	sw $a1, g # g = 3
	jal sum # call sum
	sw $v0, y # y = sum()
	lw $ra, 0($sp) # restore $ra
	addi $sp, $sp, 4 # restore $sp
	jr $ra # return to OS
sum:
	add $v0, $a0, $a1 # $v0 = a + b
	jr $ra # return
```
