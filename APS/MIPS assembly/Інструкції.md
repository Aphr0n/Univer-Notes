## Додавання

| С код      | MIPS код          |
| ---------- | ----------------- |
| а = b + c; | add $s0, $s1, $s2 |
$s0 = a, $s1 = b, $s2 = c

• add: мнемоніка (двійковий код, замінений скороченою назвою), що визначає операцію
• b, c: вхідні операнди (над якими виконується операція)
• a: цільовий операнд  (в який записується операнд)
## Bіднімання
| С код      | MIPS код          |
| ---------- | ----------------- |
| a = b - c; | sub $s0, $s1, $s2 |
$s0 = a, $s1 = b, $s2 = c
• sub: мнемоніка 
• b, c: вхідні операнди (над якими виконується операція)
• a: цільовий операнд  (в який записується операнд)
## Множення
| С код      | MIPS код      |
| ---------- | ------------- |
| a = b * c; | mult $s0, $s1 |
Множення використовує 2 регістри, тому що виконується 32×32b і результат може бути 64b що більше ніж 1 регістр, тому результат розбивається в 2 спец. регістри **`hi`** та **`lo`**
Щоб отримати результат у звичайні регістри:
```MIPS Assembly
mflo $s2   # $s2 = lo
mfhi $s3   # $s3 = hi
```

## Ділення
| С код    | MIPS код     |
| -------- | ------------ |
| a = b/c; | div $s0, $s1 |
Після виконання:
- **`lo`** → частка
- **`hi`** → залишок
Переносимо в регістри:
```MIPS Assembly
mflo $s2   # $s2 = частка
mfhi $s3   # $s3 = залишок
```
## Складені інструкції
Більш складні команди перетворюються на послідовність інструкцій MIPS.

| С код      | MIPS код    |
| ---------- | ----------- |
| а = b + c; | add a, b, c |
## Читання з пам'яті
-  Інструкція load
-  Мнемоніка: завантажити слово (lw)
-  Формат:
	`lw $s0, 5($t1)`
-  Обчислення адреси:
    -  Базова адреса ($t1) додається зі зміщенням (5)
    -  Адреса = ($t1 + 5)
-  Результат:
    -  \$s0 містить значення, що зберігається в комірці ($t1 + 5)
