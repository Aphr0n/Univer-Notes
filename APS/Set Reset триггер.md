**SR-тригер — це найпростіша логічна схема памʼяті**, яка зберігає **1 біт інформації**.
Він уміє:
- **запам’ятати 0 або 1**
- **утримувати стан**, навіть коли входи неактивні

Входи S та R використовуються для керування
- **S (Set)** — встановити `Q = 1`
- **R (Reset)** — скинути `Q = 0`

- **Q** — збережене значення
- **Q̅** — протилежне значення
