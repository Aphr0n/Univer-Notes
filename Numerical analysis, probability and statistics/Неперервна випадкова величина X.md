Випадкова величина $X$ є **неперервною**, якщо існує дійсна невід’ємна функція $f$ дійсної змінної, для якої виконується:

$$
F(x)=\int_{-\infty}^{x} f(t)\,dt
$$

для довільного $x \in \mathbb{R}$.

Функція $f$ називається **щільністю ймовірності** випадкової величини $X$, а $F$ — її **функцією розподілу**.

Для щільності ймовірності $f$ виконується:

$$
\int_{-\infty}^{\infty} f(t)\,dt = 1
$$

Основні властивості:

- $$f(x)=F'(x)$$

- $$P(X=x)=0 \quad \forall x \in \mathbb{R}$$

- $$P(a \le X < b)=F(b)-F(a)=\int_{a}^{b} f(t)\,dt$$

- Для неперервної випадкової величини:

$$
P(a \le X < b)
=
P(a < X < b)
=
P(a < X \le b)
=
P(a \le X \le b)
$$

- $$F(\infty)=\int_{-\infty}^{\infty} f(t)\,dt = 1$$

---

Графік щільності розподілу ймовірності $f$ називається **кривою розподілу ймовірності** неперервної випадкової величини $X$.
