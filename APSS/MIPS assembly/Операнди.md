Операнд — це дані або об'єкт (число, змінна, константа, вираз, функція), над яким виконується певна дія (операція), що позначається оператором (наприклад, `+`, `-`, `*`, `/`).
- Їхні значення зберігаються в
	- регістрах,
	- пам’яті або визначаються як константи (також звані прямими операндами) і потім є частиною формату інструкцій
## Операнди в регістрах 
- MIPS має 32 32-бітних регістрів
- Регістрі є швидші ніж модулі пам'ятті
- MIPS є "32-бітною архітектурою", тому що реалізує операції над 32-бітними словами
- Значення операндів регістрів:
    - Ім'я регістра починається із символу $
    - Приклад: $0, «регістр нуль», «долар нуль»
    - $0 Завжди містить **0**, його **неможливо змінити**, корисний, коли потрібен 0 в обчисленнях
    - $s0-$s7 використовуються для зберігання значень змінних, що з'являються в обчисленнях
    - $t0 - $t9 використовуються для зберігання проміжних результатів, що виникають в результаті складніших обчислень. **t** - temporary
    - Інше

| **Означення** | **Числовий код** | **Význam**                             |
| ------------- | ---------------- | -------------------------------------- |
| **$0**        | 0                | константа 0                            |
| **$at**       | 1                | Зарезервовано для псевдоінструкцій     |
| **$v0-$v1**   | 2-3              | Повернене значення з функцій           |
| **$a0-$a3**   | 4-7              | Аргументи функції                      |
| **$t0-$t7**   | 8-15             | Тимчасові змінні                       |
| **$s0-$s7**   | 16-23            | Змінні                                 |
| **$t8-$t9**   | 24-25            | Тимчасові                              |
| **$k0-$k1**   | 26-27            | Специфічні для ОС                      |
| **$gp**       | 28               | Адреса бази сегментів глобальних даних |
| **$sp**       | 29               | Вказівник стека                        |
| **$fp**       | 30               | Вказівник кадру                        |
| **$ra**       | 31               | Адреса повернення з функції            |
## Операнди в пам'яті
- 32 регістрів недостатньо для обробки великих обсягів даних
- Пам'ять має більшу ємність, але повільна порівняно з регістрами
- Часто "посилальні" операнди зберігаються в регістрах
## Константи / прямий код
- lw та sw працюють з прямим операндом (константою)
- Операнд є безпосередньо частиною IF
- Використовується 16-бітний DK
- Приклад:
- addi: додати безпосередньо
- subi: існує?


| С код                     | MIPS код                                                     |
| ------------------------- | ------------------------------------------------------------ |
| a = a + 4;<br>b = a – 12; | # $s0 = a, $s1 = b<br>addi $s0, $s0, 4<br>addi $s1, $s0, -12 |
