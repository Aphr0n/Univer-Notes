Основна ідея
Числа не мають знакового біта напряму.
Всі значення зсунуті на константу.
Формули:
```
N_MB = 2^n + N   якщо N ≥ 0
N_MB = 2^n − N   якщо N < 0
```
(еквівалентно: зберігається N + bias)
Як це працює інтуїтивно
Уяви шкалу чисел, зсунуту вправо:
```
реальні:   -3  -2  -1   0   1   2   3
кодові:     1   2   3   4   5   6   7
```
Тут 0 кодується не як 0000, а як середина діапазону.
Приклад (4 біти, bias = 8)
Реальне	Код
```
−8	0000
−1	0111
0	1000
+7	1111
```
