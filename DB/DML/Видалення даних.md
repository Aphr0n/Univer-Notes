# DELETE
Команда `DELETE` видаляє рядок з таблиці. Є транзакційною операцією

## Синтаксис
```SQL
DELETE FROM tabulka
[WHERE podmienka]
```
## Приклад
```SQL
DELETE FROM student
WHERE id=3
```

# TRUNCATE
Команда `TRUNCATE` видаляє *всі* рядки з таблиці
## Синтаксис
```SQL
TRUNCATE TABLE tabulka
```
## Приклад
```SQL
TRUNCATE TABLE student
```
