Перевантаження(Overload) - створення декількох методів з одним і тим ж самим ім'ям, але з різними параметрами. Вибір конкретного методу відбувається **під час компіляції** (статично) залежно від порядку передачі аргументу. Перевантаження (і перевизначення разом) може бути надмірним, тому слід уникати перевантаження, коли це можливо
```Java
public class Utility {
    public String whatAmI(Object o) { return "object"; }
    public String whatAmI(String s) { return "string"; }
}

```
- Перевантажує, але НЕ перевизначає успадкований метод boolean isString(Object o) з класу Utility
```Java
public static void main(String[] args) {
	Overloaded overloaded = new Overloaded();
	System.out.println(overloaded.whatAmI(overloaded)); //Object
	System.out.println(overloaded.whatAmI(new String("test"))); //string 
	Object o = new String("test");
	System.out.println(overloaded.whatAmI(o)); //object
	System.out.println(overloaded.isString(new String("test"))); //true
	System.out.println(overloaded.isString(o)); //false
	Utility utility = overloaded;
	System.out.println(utility.isString(new String("test"))); //false
}
```
