- Тип R(egister)
- 3 операнди
	- rs, rt: регістри джерела
	- rd: цільовий регістр
- Додаткові поля:
	- **op:** код операції (**0 для інструкцій типу R**)
	- **funct:** поле для визначення **конкретного типу інструкції**
	- **shamt:** для **операцій зсуву** задає кількість зсувів, **інакше 0**
