| Назва представлення     | Інша назва                | Опис                                                        | Діапазон для n біт     |
| ----------------------- | ------------------------- | ----------------------------------------------------------- | ---------------------- |
| Прямий код              | Sign-Magnitude            | Старший біт — знак (0 «+», 1 «−»), інші біти — модуль числа | ±(2^(n−1) − 1)         |
| Інверсний код           | One’s Complement          | Від’ємне число отримується інверсією всіх бітів додатного   | ±(2^(n−1) − 1)         |
| Доповняльний код        | Two’s Complement          | Від’ємне число = інверсія + 1; найпоширеніше представлення  | −2^(n−1) … 2^(n−1) − 1 |
| Зміщений код            | Bias / Excess             | Значення зберігається зі зміщенням (bias)                   | Залежить від bias      |
| Код із фіксованою комою | Fixed-Point               | Двійкове число з фіксованим положенням коми                 | Залежить від формату   |
| Код із рухомою комою    | Floating-Point (IEEE 754) | Представлення чисел як знак, порядок і мантиса              | Дуже широкий           |
| BCD                     | Binary-Coded Decimal      | Кожна десяткова цифра кодується 4 бітами                    | 0–9 на цифру           |
| Код Грея                | Gray Code                 | Сусідні значення відрізняються лише 1 бітом                 | Не для арифметики      |
| Q-формат                | Q-format                  | Фіксована кома: Qm.n — m біт цілої, n біт дробової частини  | Залежить від m, n      |
