SQL - стандартизована мова, призначена для взаємодії з реляційною базою даних SRBD для зберігання, маніпулювання та отримання даних у реляційних базах даних. 
- Непроцедурна (декларативна) мова. 
	- Що ми хочемо отримати, а не як ми хочемо отримати дані (це не послідовність кроків). 
	- SRBD отримує дані з бази даних (фізичного сховища). 
- Заснована на реляційному численні кортежів.

# Типи команд SQL
- DDL (Data Definition Language) – мова для визначення структури бази даних, включаючи зміни до структури – CREATE, DROP, ALTER, TRUNCATE. 
- DML (Data Manipulation Language) – мова для маніпулювання даними, включаючи запити – SELECT, INSERT, UPDATE, DELETE. 
- DCL (Data Control Language) – мова для керування доступом до даних – GRANT, REVOKE. 
- TCL (Transaction Control Language) – мова для керування транзакціями – COMMIT, ROLLBACK, BEGIN WORK.
