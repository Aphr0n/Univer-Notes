# Загальний підхід
Під час розв’язування рівняння f(x)=0 можна використати такий алгоритм:
1. Визначити інтервал (a,b), у якому містяться всі дійсні корені рівняння.
2. Визначити кількість коренів, що лежать в інтервалі (a,b).
3. Знайти інтервали $(a_i, b_i)$ такі, що $f(c_i) = 0$, де  $c_i \in (a_i, b_i)$ — i-тий корінь рівняння, і для кожного  $x \in (a_i, b_i)$ виконується $f(x) \neq 0$ при $x \neq c_i$​. (Цей крок називається **відокремленням коренів**.)
4. Застосувати наближені методи для обчислення коренів рівняння.
5. Оцінити абсолютну похибку обчисленого кореня.

# Оцінка абсолютної похибки
Для оцінки абсолютної похибки наближеного значення $x^{(*)}$ кореня c можна використати таку теорему:
### **Теорема**
Нехай c — точне значення кореня, а $x^{(*)}$ — його наближене значення.  
Нехай обидва значення належать інтервалу (a,b) і $|f'(x)| \ge m > 0$ для всіх $x \in ⟨a, b⟩$
Тоді справедлива оцінка:
$|c - x^{(*)}| \le \frac{|f(x^{(*)})|}{m}$​

# Графічний метод
Нехай задано рівняння f(x) = 0. Якщо побудувати графік функції f, то **x-координати точок перетину графіка функції** ) y = f(x) з віссю x є дійсними коренями рівняння (1).
Знайти поведінку функції f та її властивості загалом може бути непросто, тому, якщо це можливо, замінимо рівняння (1) еквівалентним рівнянням g(x) = h(x), де функції g і h є простішими, а їхні графіки легше накреслити. Тоді дійсні корені рівняння (1) відповідають **x-координатам точок перетину графіків функцій**  
y = g(x) та y=h(x). Такий підхід доцільно використовувати під час відокремлення (сепарації) коренів рівняння.
