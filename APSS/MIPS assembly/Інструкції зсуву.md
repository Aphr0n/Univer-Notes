- sll: логічний зсув вліво на фіксовану кількість бітів
	- Приклад: sll $t0, $t1, 5 # $t0 <= $t1 << 5
- srl: логічний зсув вправо на фіксовану кількість бітів
	- Приклад: srl $t0, $t1, 5 # $t0 <= $t1 >> 5
- sra: арифметичний зсув вправо на фіксовану кількість бітів
	- Приклад : sra $t0, $t1, 5 # $t0 <= $t1 >>> 5
- sllv: логічний зсув вліво на змінну кількість бітів
	- Приклад: sllv $t0, $t1, $t2 # $t0 <= $t1 << $t2
- srlv: логічний зсув вправо на змінну кількість бітів
	- Приклад: srlv $t0, $t1, $t2 # $t0 <= $t1 >> $t2
- srav: арифметичний зсув вправо на змінну кількість бітів
	- Приклад: srav $t0, $t1, $t2 # $t0 <= $t1 >>> $t2
