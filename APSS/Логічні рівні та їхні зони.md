Для безпечного сприйняття сигналу електроніка використовує **діапазони напруг для входу та виходу**:


| Параметр          | Значення                                   |
| ----------------- | ------------------------------------------ |
| VOH (Output High) | мінімальна напруга, яку вихід видає як 1   |
| VOL (Output Low)  | максимальна напруга, яку вихід видає як 0  |
| VIH (Input High)  | мінімальна напруга, яку вхід трактує як 1  |
| VIL (Input Low)   | максимальна напруга, яку вхід трактує як 0 |
**Forbidden zone (заборонена зона)** — це проміжок між VIL і VIH.  
Якщо сигнал потрапляє сюди, логіка може працювати нестабільно.
