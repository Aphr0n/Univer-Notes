# CHECK
```SQL
CREATE TABLE s u d e n t (
. . .
year INT NOT NULL
CHECK(year >=1 AND year <=5)
) ;
```

# ALTER TABLE
Змінює структуру таблиці:
- шляхом додавання (ADD) стовпця,
- шляхом видалення (DROP COLUMN) стовпця або
- шляхом зміни (ALTER COLUMN) стовпця.
Синтаксис:
```SQL
ALTER TABLE таблиця a
операція [ тип ]
[ обмеження ] ;
```

Приклад:
- Додати стовпець:
```SQL
	ALTER TABLE student ADD sex CHAR;
```
- Видалити стовпець:
```SQL
	ALTER TABLE student DROP COLUMN sex;
```
- Змінити стовпець:
```SQL
	ALTER TABLE student
	ALTER COLUMN name TYPE VARCHAR(20);
```
