DROP - видаляє таблицю
```SQL
DROP TABLE DBSUser ;
```

# Превентивне видалення
Видаляє попередню версію таблиці (або представлення) перед створенням нової. Розміщується перед фактичним створенням таблиці (або представлення). Забезпечує доступність імені.

```SQL
DROP TABLE I F EXISTS nazov ;
CREATE TABLE nazov . . . ;
```
