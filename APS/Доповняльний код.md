Доповняльний код — це спосіб подання знакових цілих чисел, який усуває всі недоліки прямого коду.

Для N біт:
$[ знаковий біт ][ решта бітів числа ]$

Найстарший біт:
0 → додатне число
1 → від’ємне число

---
Коли старший біт = 1, то треба інвертувати всі біти числа

Приклад:
```
1010
```
Інверсія:
```
0101
```
Потім +1:
```
0110₂ = 6₁₀
```
Отже:
```
1010₂ = −6₁₀
```
