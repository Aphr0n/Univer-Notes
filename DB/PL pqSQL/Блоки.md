Базовою одиницею в будь-якому коді PL/pgSQL є блок. Весь код PL/pgSQL складається з одного блоку або кількох блоків, які записуються послідовно та/або вкладені
в інший блок.
Існує два типи блоків:
- Анонімні блоки.
- Створюються динамічно та виконуються користувачем лише один раз.
- Іменовані блоки (функції та збережені процедури).
	- Вони мають призначене ім'я,
	- зберігаються в базі даних,
	- можуть виконуватися багаторазово,
	- можуть містити параметри.

# Анонімні блоки
## Структура
```SQL
DO $$
[ <<label>>]
DECLARE
/* Declare section (optional) */
BEGIN
/* Execute section (required). */
EXCEPTION
/* Exception handling section (optional). */
END [label]
$$ ;
```
