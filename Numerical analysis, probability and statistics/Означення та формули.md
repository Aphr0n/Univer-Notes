| Символ          | Назва змінної                  | Що означає (тлумачення)                                                |
| --------------- | ------------------------------ | ---------------------------------------------------------------------- |
| $n$             | Обсяг вибірки                  | Загальна кількість об'єктів або вимірювань у дослідженні.              |
| $x_i$           | Значення ознаки                | Конкретне числове значення, виміряне у _i_-ї одиниці.                  |
| $n_i$ ($n_k$)   | Абсолютна частота              | Кількість разів, яку конкретне значення зустрічається у вибірці.       |
| $f_i$ ($f_k$)   | Відносна частота               | Частка конкретного значення від загального обсягу вибірки ($n_i / n$). |
| $N_i$ ($N_k$)   | Кумулятивна частота            | Накопичена сума частот від початку до поточного значення.              |
| $\bar{x}$       | Вибіркове середнє              | Середнє арифметичне всіх значень вибірки; центр даних.                 |
| $s^2$           | Вибіркова дисперсія            | Міра того, наскільки сильно дані «розкидані» навколо середнього.       |
| $s$             | Середньоквадратичне відхилення | Типова відстань значень від середнього (корінь із дисперсії).          |
| $R$             | Варіаційний розмах             | Різниця між найбільшим і найменшим значеннями ($x_{max} - x_{min}$).   |
| $\hat{x}$       | Мода                           | Значення, яке зустрічається у вибірці найчастіше.                      |
| $\tilde{x}$     | Медіана                        | Значення, що стоїть рівно посередині впорядкованого ряду.              |
| $\gamma_3$      | Коефіцієнт асиметрії           | Показник того, чи зміщені дані в один бік від середнього.              |
| $\gamma_4$      | Коефіцієнт ексцесу             | Показник того, наскільки «гострою» є вершина розподілу даних.          |
| $E(X)$          | Математичне сподівання         | Теоретичне середнє значення випадкової величини.                       |
| $D(X)$          | Дисперсія випадкової величини  | Теоретичний розкид випадкової величини.                                |
| $\alpha$        | Рівень значущості              | Ймовірність помилитися, відхиливши правильну нульову гіпотезу.         |
| $r$ ($r_{x,y}$) | Коефіцієнт кореляції Пірсона   | Сила та напрямок лінійного зв'язку між двома змінними.                 |
| $R^2$           | Коефіцієнт детермінації        | Частка мінливості даних, яку пояснює побудована модель.                |
| $H_0$           | Нульова гіпотеза               | Твердження про відсутність різниці або ефекту, яке ми перевіряємо.     |
| $H_1$           | Альтернативна гіпотеза         | Твердження, яке ми приймаємо, якщо нульова гіпотеза хибна.             |
# 📊 Статистика та ймовірність — формули

## 1. Описова статистика (характеристики вибірки)
>[!info] $\frac{1}{n-1}$ для вибірки

| Формула                                     | Що рахує              | Тлумачення                      |
| ------------------------------------------- | --------------------- | ------------------------------- |
| $f_i = \frac{n_i}{n}$                       | Відносна частота      | Частка появи значення у вибірці |
| $\bar{x} = \frac{1}{n}\sum x_i$             | Арифметичне середнє   | Центр даних                     |
| $\bar{x}_g = \sqrt[n]{\prod x_i}$           | Геометричне середнє   | Середній темп росту             |
| $s^2 = \frac{1}{n-1}\sum (x_i - \bar{x})^2$ | Дисперсія             | Розкид даних                    |
| $s = \sqrt{s^2}$                            | Стандартне відхилення | Середня відстань від середнього |
| $\gamma_3 = \frac{\mu_3}{s^3}$              | Асиметрія             | Скошеність розподілу            |
| $\gamma_4 = \frac{\mu_4}{s^4} - 3$          | Ексцес                | Гостровершинність               |

## 2. Випадкові величини та ймовірність

| Формула | Що рахує | Тлумачення |
|---|---|---|
| $E(X) = \sum x_j p_j$ | Математичне сподівання | Теоретичне середнє |
| $D(X) = E(X^2) - [E(X)]^2$ | Дисперсія | Теоретичний розкид |
| $Z = \frac{X - E(X)}{\sqrt{D(X)}}$ | Нормування | Стандартизація |
| $P(X=x) = \binom{n}{x} p^x (1-p)^{n-x}$ | Біноміальний розподіл | Серія успіхів |
| $P(X=x) = \frac{\lambda^x e^{-\lambda}}{x!}$ | Пуассон | Рідкісні події |

## 3. Кореляція та регресія

| Формула                                                     | Що рахує     | Тлумачення       |
| ----------------------------------------------------------- | ------------ | ---------------- |
| $cov(x,y) = \frac{1}{n}\sum (x_i - \bar{x})(y_i - \bar{y})$ | Коваріація   | Спільна зміна    |
| $r_{x,y} = \frac{cov(x,y)}{s_x s_y}$                        | Пірсон       | Лінійний зв’язок |
| $r_s = 1 - \frac{6\sum D_i^2}{n(n^2-1)}$                    | Спірмен      | Ранговий зв’язок |
| $\hat{Y} = a_0 + a_1 x$                                     | Регресія     | Прогноз          |
| $R^2 = \frac{SSR}{SST}$                                     | Детермінація | Якість моделі    |

## 4. Перевірка гіпотез

| Формула                                       | Що рахує    | Коли застосовується  |
| --------------------------------------------- | ----------- | -------------------- |
| $Z = \frac{\bar{x} - \mu_0}{\sigma/\sqrt{n}}$ | Z-критерій  | Відоме σ             |
| $t = \frac{\bar{x} - \mu_0}{s/\sqrt{n}}$      | t-критерій  | σ невідоме           |
| $\chi^2 = \frac{(n-1)s^2}{\sigma_0^2}$        | χ²-критерій | Дисперсія            |
| $F = \frac{s_1^2}{s_2^2}$                     | F-критерій  | Порівняння дисперсій |
