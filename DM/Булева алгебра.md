Алгебраїчна система (L,⊻,⊼,′,O,I) є *булевою алгеброю* якщо L≠∅,⊻,⊼ — бінарні операції, а ′ — унарна операція на L, і виконуються такі властивості:
- ідемпотентність,  
- комутативність,  
- асоціативність,  
- закон поглинання,  
- дистрибутивність  
- існування доповнення для кожного елемента з LLL.
Будь яку формулу з булевої алгебри можна подати мовою висловлюваної логіки і навпаки
## Операції

| №    | Закон                      | Пояснення                                 |
| ---- | -------------------------- | ----------------------------------------- |
| (1)  | \( x ⊻ O = x \)            | «Або» з мінімумом нічого не змінює        |
| (2)  | \( x ⊼ I = x \)            | «І» з максимумом нічого не змінює         |
| (3)  | \( x ⊻ x′ = I \)           | Елемент або його заперечення дає максимум |
| (4)  | \( x ⊼ x′ = O \)           | Елемент і його заперечення дає мінімум    |
| (5)  | \( x ⊼ O = O \)            | «І» з мінімумом дає мінімум               |
| (6)  | \( x ⊻ I = I \)            | «Або» з максимумом дає максимум           |
| (7)  | \( (x ⊻ y)′ = x′ ⊼ y′ \)   | Перше правило де Моргана                  |
| (8)  | \( (x ⊼ y)′ = x′ ⊻ y′ \)   | Друге правило де Моргана                  |
| (9)  | \( x ⊻ (x′ ⊼ y) = x ⊻ y \) | Спрощення: x ⅴ (!x ∧ y) = x ⅴ y           |
| (10) | \( x ⊼ (x′ ⊻ y) = x ⊼ y \) | Спрощення: x ∧ (!x ⅴ y) = x ∧ y           |

## Булева функція
Нехай (D,⊻,⊼,′,0,1)(D, ⊻, ⊼, ′, 0, 1)(D,⊻,⊼,′,0,1) — булева алгебра, де D={0,1}D = \{0, 1\}D={0,1}.  
Відображення f:Dn→Df : D^n → Df:Dn→D називається *булевою функцією* з nnn змінними.  
Її записують так:
y=f(x1,x2,…,xn),
де y∈D і $x_i$∈D для всіх i=1,2,…,n

**Твердження:**  
Нехай (D,⊻,⊼,′,0,1) — булева алгебра.  
Тоді існує рівно $2^{2^n}$ різних булевих функцій
y=f(x1,x2,…,xn)
з n змінними.

## Висловлювальна логіка ↔ Булева алгебра
Інтерпретацією мови висловлювальної логіки (Ω,F) у булевій алгебрі B=(D, ∨, ∧, ′, 0, 1) називається відображення ph:F→{0,1} таке, що виконуються такі рівності:
$ph(¬A)=(ph(A))′,$
$ph(A∨B)=ph(A) ∨ ph(B),$
$ph(A∧B)=ph(A) ∧ ph(B),$
$ph(A⇒B)=(ph(A))′ ∨ ph(B),$
$ph(A⇔B)=(ph(A) ∧ ph(B)) ∨ ((ph(A))′ ∧ (ph(B))′).$

Значення ph(A)ph(A)ph(A) називається **істиннісним значенням формули** A∈F
