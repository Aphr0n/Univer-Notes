Це розширення звичайного `IF/ELSE` — дозволяє перевіряти кілька умов по черзі без вкладення.

```SQL
IF number = 0 THEN
    result := 'zero';

ELSIF number > 0 THEN
    result := 'positive';

ELSIF number < 0 THEN
    result := 'negative';

ELSE
    result := 'NULL';  -- якщо number IS NULL
END IF;
```

---
## Логіка роботи

- Умови перевіряються зверху вниз.
- Виконується перша умова, яка виявилась істинною.
- `ELSE` виконується, якщо жодна умова не спрацювала.

# CASE — дві форми

## 1. Простий CASE (`Simple CASE`)

Порівнює одне значення з набором варіантів  
(аналог `switch` у Java).

```sql
CASE grade
    WHEN 'A' THEN
        result := 'Excellent';

    WHEN 'B' THEN
        result := 'Good';

    WHEN 'C' THEN
        result := 'Average';

    ELSE
        result := 'Unknown';
END CASE;
```

### Особливість

`grade` обчислюється один раз, після чого порівнюється з кожним `WHEN`.

---
## 2. Пошуковий CASE (`Searched CASE`)

Перевіряє довільні булеві вирази  
(більш гнучкий варіант).

```sql
CASE
    WHEN score >= 90 THEN
        result := 'A';

    WHEN score >= 75 THEN
        result := 'B';

    WHEN score >= 60 THEN
        result := 'C';

    ELSE
        result := 'F';
END CASE;
```

---

## Логіка роботи

- Умови перевіряються по черзі.
- Виконується перша істинна умова.
- Це аналог `IF / ELSIF`, але синтаксично чистіший.
# Цикли
## LOOP

LOOP - це нескінченний цикл, який виконується до тих пір, поки не зустріне явний EXIT.
```sql

LOOP
    -- тіло циклу
    EXIT WHEN умова;   -- вихід якщо умова true
END LOOP;

Або з IF:
sql

LOOP
    i := i + 1;
    IF i > 10 THEN
        EXIT;
    END IF;
END LOOP;
```
Також можна використовувати з міткою, якщо цикли вкладені:
```sql
<<outer>>
LOOP
    LOOP
        EXIT outer WHEN i > 10;  -- виходить із зовнішнього циклу
    END LOOP;
END LOOP;
```

## `CONTINUE`

Аналог `continue` в Python/Java — пропускає решту тіла циклу і переходить до наступної ітерації.

```sql
CONTINUE;                     -- наступна ітерація найвнутрішнього циклу
CONTINUE my_loop;             -- наступна ітерація циклу з міткою my_loop
CONTINUE WHEN i > 5;         -- тільки якщо умова true
```

---

## `WHILE`

Класичний цикл з **передумовою** — виконується поки умова `true`.

```sql
WHILE i < 10 LOOP
    i := i + 1;
END LOOP;
```

---

## `FOR` — по діапазону цілих

Змінна `i` створюється автоматично і живе лише всередині циклу.

```sql
FOR i IN 1..10 LOOP           -- від 1 до 10, крок 1
    ...
END LOOP;

FOR i IN REVERSE 10..1 LOOP  -- від 10 до 1
    ...
END LOOP;

FOR i IN 1..10 BY 2 LOOP     -- крок 2: 1, 3, 5, 7, 9
    ...
END LOOP;
```

---

## `FOR` — по результату запиту

Ітерація по рядках результату `SELECT`. На кожній ітерації змінна типу `record` містить один рядок таблиці, поля доступні через крапку.

```sql
DECLARE
    zaznammiest record;
BEGIN
    FOR zaznammiest IN
        SELECT nazov, populacia FROM mesto
    LOOP
        RAISE NOTICE 'Mesto % ma % obyvatelov.',
            zaznammiest.nazov,
            zaznammiest.populacia;
    END LOOP;
END;
```

> `RAISE NOTICE` — вивід повідомлення (аналог `print`), де `%` — плейсхолдери для значень.
