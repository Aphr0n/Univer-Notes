Припустімо, що рівняння  **f(x) = 0**  має рівно один корінь **c** на інтервалі **(a, b)**.
Визначимо послідовність інтервалів ⟨aₖ, bₖ⟩ і значень cₖ для *k = 0, 1, 2, …* так:
### K₀
Початковий інтервал:
- ⟨a₀, b₀⟩ = ⟨a, b⟩  
- причому **f(a₀) · f(b₀) < 0** (бо **f(a) · f(b) < 0**)

Середина інтервалу:
```text
c₀ = (a₀ + b₀) / 2
```

Перевірка:
- якщо **f(c₀) ≠ 0**, переходимо до кроку **K₁**;
- інакше — кінець, **c₀** є коренем.
### K₁
Оновлення інтервалу:
- ⟨a₁, b₁⟩ = ⟨a₀, c₀⟩, якщо **f(a₀) · f(c₀) < 0**,  
  **або**
- ⟨a₁, b₁⟩ = ⟨c₀, b₀⟩, якщо **f(c₀) · f(b₀) < 0**.

Середина нового інтервалу:
```text
c₁ = (a₁ + b₁) / 2
```

Перевірка:
- якщо **f(c₁) ≠ 0**, переходимо до кроку **$K_n$**;
- інакше — кінець, **c₁** є коренем.
### Загальний крок Kₖ (k ≥ 1)
Оновлення інтервалу:
- ⟨aₖ, bₖ⟩ = ⟨aₖ₋₁, cₖ₋₁⟩, якщо **f(aₖ₋₁) · f(cₖ₋₁) < 0**,  
  **або**
- ⟨aₖ, bₖ⟩ = ⟨cₖ₋₁, bₖ₋₁⟩, якщо **f(cₖ₋₁) · f(bₖ₋₁) < 0**.

Середина нового інтервалу:
```text
cₖ = (aₖ + bₖ) / 2
```

Перевірка:
- якщо **f(cₖ) ≠ 0**, переходимо до кроку **Kₖ₊₁**;
- інакше — кінець, **cₖ** є коренем.
# Оцінка помилки
Pre odhad $x^{(*)} = c_k$, absolútnej chyby koreňa c platí:
$$|x^{(*)} - c| \underline\leq \frac{b-a}{2^{(k+1)}} = \overline{ER}(x^{(*)})$$

### Примітка
На кожному кроці довжина інтервалу зменшується вдвічі, а корінь **c** завжди залишається всередині ⟨aₖ, bₖ⟩.
