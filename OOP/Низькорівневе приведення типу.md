Низькорівневе приведення типу(Downcasting) - процес перетворення **посилання на базовий клас** у **посилання на його похідний клас**. Є небезпечним, бо не перевіряється на етапі компіляції. 
```Java
Article a = new DiscountArticle(); // a насправді є DiscountArticle
DiscountArticle da = (DiscountArticle) a; // downcasting можливий
```

```Java
Object o = new Object();
String s = (String) o; // ClassCastException під час виконання
```
Для уникнення ClassCastException слід використати `instanceof`, або принцип `LSP`
```Java
Object o = ...;
if (o instanceof String) {
    String s = (String) o;
}
```

```Java
Class java.util.Vector has reusable methods
public void addElement(Object obj)
public Object elementAt(int i)
public class StringSet {
	private Vector elements;
	public void insert(String s) {
		elements.addElement(s);
	}
	public String choose() {
		return elements.elementAt(0);
		return (String) elements.elementAt(0);
	}
}
```
