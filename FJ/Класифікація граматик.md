	# G(3) - Регулярні граматики
## Дозволені форми

### Праволінійна граматика

```
A → aB
A → a
A → ε
```

### Ліволінійна граматика
```
A → Ba
A → a
A → ε
```
## Головна умова
### Справа:
- максимум один нетермінал
- і він лише в кінці(праволінійна граматика) або спочатку(ліволінійна)
### Зліва
має бути:
- Лише 1 нетермінал
# G(2) - Безконтекстова граматика
## Правило
### Зліва
має бути:
```
ОДИН нетермінал
```
### Справа
може бути будь-що:
- термінали
- нетермінали
- змішані

# G(1) - Контекстова граматика
Нетермінал `A` можна переписати:
```
лише якщо навколо певний контекст
```
# Найважливіша умова
Правило не повинно скорочувати рядок:

$$|\alpha| \le |\beta|$$
тобто:
```
довжина правої частини ≥ довжини лівої
```

# G(0) - Фразова граматика
Найзагальніші.
## Правило
```
α → β
```
де в `α` є хоча б один нетермінал.
# Тут майже нема обмежень

Можна:

- скорочувати рядок
- мати будь-які конструкції
- будь-який контекст
