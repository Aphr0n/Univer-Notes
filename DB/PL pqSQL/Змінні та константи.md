Змінні використовуються для:
- Тимчасового зберігання даних.
- Маніпуляції збереженими значеннями.
- Повторного використання.
Оголошуються в декларативній секції всередині блоку.
Оголошені змінні ініціалізуються своїми значеннями за замовчуванням щоразу, коли викликається блок, а не лише один раз, коли викликається функція.
Змінні в декларативній секції можуть перевантажувати змінні з тим самим іменем, що й у зовнішньому блоці.
Якщо зовнішній блок названо міткою, то його змінні доступні всередині вкладеного блоку за допомогою:
```sql
<label>.<varname>
```

За допомогою оператор SELECT INTO можна вибирати дані з бази даних та присвоювати ці дані змінній.
```SQL
select select_list
into variable_name
from table_expression;
```
## Синтаксис
```SQL
identifier [CONSTANT] datatype [NOT NULL]
[ : = | = | DEFAULT expr ] 
```

## Приклади
```SQL
DECLARE
v_birthday DATE;
v_age INT NOT NULL = 2 1 ;
v_name VARCHAR( 1 5 ) := ’ Homer ’ ;
v_magic CONSTANT NUMERIC := 4 2 ;
v_valid BOOLEAN DEFAULT TRUE;
```

# Константи
На відміну від змінної, значення константи не може бути змінене після її ініціалізації.

## Синтаксис
```SQL
constant_name constant data.type := expression;
```
