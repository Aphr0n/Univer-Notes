Дано таку послідовність інструкцій:

```
	addi $s0, $0, 4
	addi $s1, $0, 1
	sll $s1, $s1, 2
	beq $s0, $s1, target
	addi $s1, $s1, 1
target:
	addi $s1, $s1, 2
```

Яке значення буде мати $s1?

Розв'язок
1.  Читаємо рядки наданого MIPS коду і рахуємо як буде змінюватися регістр
	1. `addi $s0, $0, 4` - регістр 0 отримав значення 4
	2. `addi $s1, $0, 1` - регістр 0 отримав значення 1
	3. `sll $s1, $s1, 2` - регістр 1 логічно зсунувся на 2 (0001 <- 0100)
	4. `beq $s0, $s1, target` - beq означає перехід по мітці target якщо $s0 == $s1, умова виконалася тому переходимо по мітці
	5. `addi $s1, $s1, 2` - 4 + 2 = 6
