**Зсувний функціональний блок** — це апаратний вузол, який виконує операції над бітами слова без арифметичних обчислень:
- логічний зсув вліво / вправо
- арифметичний зсув
- циклічний зсув (rotation)
Ці операції є стандартною частиною набору інструкцій більшості процесорів.

### Логічний зсув
**Логічний зсув** — це бітова операція, під час якої всі біти двійкового числа зміщуються в один бік, а вивільнені позиції заповнюються нулями. Використовується при швидкому множенні
A << N = A × 2N
- 00001 << 2 = 00100 (1 × 22 = 4)
-11101 << 2 = 10100 (-3 × 22 = -12)
A >>> N = A ÷ 2N
- 01000 >>> 2 = 00010 (8 ÷ 22 = 2)
- 10000 >>> 2 = 11100 (-16 ÷ 22 = -4)

```
00101101  (45)
<< 2
---------
10110100  (180)
```
### Арифметичний зсув
**Арифметичний зсув** — це бітова операція, яка працює зі **знаковими числами** і намагається зберегти їх числовий зміст під час зсуву.
- біти зсуваються вправо
- **старші біти заповнюються копіями біта знаку**
- молодші біти відкидаються
```
11101000  (-24)
>> 2
---------
11111010  (-6)
```

### Циклічний зсув
Циклічний зсув - це бітова операція, яка "циклічно обертає" число
- біти рухаються в один бік
- жоден біт не зникає
- те, що «випало» з одного краю, **повертається з іншого**
```
10010110  (8 біт)
ROL 1
↓
00101101
```
