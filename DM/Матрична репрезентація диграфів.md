# Матриця інцидентності
Нехай $\vec{G} = (V, \vec{H})$ — орієнтований граф (диграф), де  
$V = \{v_1, v_2, \dots, v_n\}$,  
$\vec{H} = \{h_1, h_2, \dots, h_m\}$
Матрицею інцидентності дiграфа $\vec{G}$ називається матриця  
$A = \{a_{ij}\}$ розміру n×m, якщо для її елементів виконується:

$$
a_{ij} =
\begin{cases}
1, & \text{якщо } h_j = (v_i, v_k),\ v_k \in V \\
-1, & \text{якщо } h_j = (v_k, v_i),\ v_k \in V \\
0, & \text{інакше}
\end{cases}
$$
**Примітка. Основні властивості матриці AAA:**

- у кожному стовпці числа 1 і −1 зустрічаються рівно один раз, решта елементів — нулі;
- кількість одиниць у i-му рядку дорівнює вихідному степеню вершини $\deg^+(v_i)$
- кількість мінус одиниць у i-му рядку дорівнює вхідному степеню вершини $\deg^-(v_i)$

## Матриця сусідності
Нехай $\vec{G} = (V, \vec{H})$ — діграф, де  $V = {v_1, v_2, \dots, v_n}$.

Матрицею суміжності діграфа $\vec{G}$ називається квадратна матриця  
$B = {b_{ij}}$ розміру $n \times n$, якщо для її елементів виконується:
$$
b_{ij} =
\begin{cases}
1, & \text{якщо } \vec{h} = (v_i, v_j) \in \vec{H} \\
0, & \text{інакше}
\end{cases}
$$
# Теореми
Нехай $\vec{G} = (V, \vec{H})$ — діграф, а $B$ — його матриця суміжності.  
Тоді для кожного $n \in \mathbb{N}$ елемент $b^{(n)}_{ij}$ матриці $B^n$ дорівнює кількості шляхів довжини $n$ з вершини $v_i$ до вершини $v_j$.
$$
B^n = \left( b^{(n)}_{ij} \right), \quad
b^{(n)}_{ij} = \text{кількість шляхів довжини } n
\text{ з } v_i \text{ до } v_j
$$
