# Машина Тюрінга — компоненти

- нескінченна стрічка (пам’ять)
- головка читання/запису
- скінченний керуючий автомат

## Характеристика

- проста структура
- надзвичайно висока виразність

>[!info] Зауваження Навіть дуже складні програми можна звести до цієї моделі.

# Формальне означення машини Тюрінга

```text
M = (Q, Σ, Γ, δ, q0, qacc, qrej)
```


## Функція переходів

```text
δ : Q × Γ → Q × Γ × {L, R}
```

## Крок роботи

- читання символу  
- запис нового символу  
- рух головки  
- зміна стану  

Кожен крок є локальним, але весь обчислювальний процес — глобальний.


# Конфігурації та обчислення

## Конфігурація

```text
(стан, вміст стрічки, позиція головки)
```

>Обчислення це послідовність конфігурацій.

## Завершення роботи

Можливі варіанти:

- прийняття (`qacc`)
- відхилення (`qrej`)
- нескінченна робота

>[!info] Не кожен алгоритм обов’язково завершується.

# Типи мов для машини Тюрінга

## Розв’язні мови

- машина Тюрінга **завжди зупиняється**
- завжди видає відповідь «так» або «ні»

## Напіврозв’язні (RE)

- машина Тюрінга може зупинитися лише при прийнятті
- при відхиленні може зациклитися

## Різниця

- **розв’язні:** завжди є кінцевий результат  
- **RE:** результат може не з’явитися взагалі


>[!info] Не всі задачі мають алгоритмічне розв’язання.


# Теза Чьорча–Тюрінга
