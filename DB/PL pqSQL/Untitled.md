SQL не є повним за Тюрінгом, тобто він не має виразної сили машини Тюрінга.
PL/pgSQL дозволяє писати імперативний код.
- Керувати конструкціями циклів, розгалужень тощо.
- Визначати та виконувати програмні блоки.

Процедурна мова – розширення мови структурованих запитів.
- Oracle PL/SQL,
- Microsoft SQL Server використовує Transact-SQL.
PostgreSQL наразі підтримує кілька стандартних процедурних мов. PL/pgSQL, PL/Tcl, PL/Perl, PL/Python, PL/Java та багато інших.
- PostgreSQL за замовчуванням використовує PL/pgSQL.
